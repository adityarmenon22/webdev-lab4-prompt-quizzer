This is a small JavaScript quiz game for the Web Development I Lab Assignment 4. It runs completely in the browser console using `prompt()` for input and `alert()` for output. The quiz asks multiple questions, checks the answers, and shows the final score at the end. [file:1]

## Features

- Questions and answers stored in an array of objects  
- Uses a `for` loop to go through all questions one by one  
- Compares user input after `toLowerCase()` and `trim()` for better checking  
- Shows feedback after every question (correct / wrong)  
- Displays total score when the quiz finishes [file:1]
